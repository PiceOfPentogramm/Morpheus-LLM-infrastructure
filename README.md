# Morpheus-LLM-infrastructure

## Архитектура проекта

Проект разделен на две части, которые общаются через API:

### Инфраструктура (Infrastructure)
- Интерфейс настройки (GUI для первоначальной настройки)
- Подключение к Telegram боту
- API сервер для связи с ядром
- Упаковка в .exe файл

### Ядро (Core)
- Логика работы AI модели
- Обработка запросов через API
- На этапе MVP - простая заглушка

## Как это работает
1. Пользователь запускает .exe на своем компьютере
2. Открывается окно настройки (токен бота, параметры)
3. После настройки вся работа идет через Telegram
4. Инфраструктура получает сообщения из Telegram → отправляет в API → Ядро обрабатывает → возвращает ответ → отправляет в Telegram

## Этапы разработки
1. **Сначала**: Инфраструктура с заглушкой для MVP
2. **Потом**: Полноценное ядро с AI логикой
